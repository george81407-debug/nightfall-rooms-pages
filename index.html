Nightfall Rooms Uploader

Nightfall Rooms Uploader is a private, internal automation tool built and used solely by the owner/operator of the Nightfall Rooms YouTube channel to manage and publish that channel's content. It is not a public-facing product or service, and it is not available for use by any other party.

What this application does

Using the YouTube Data API and YouTube Analytics API via Google OAuth 2.0, the application:

  • Uploads long-form ambience videos to the operator's own Nightfall Rooms YouTube channel
  • Sets each video's title, description, tags, privacy status, and AI/synthetic-content disclosure flag
  • Reads the operator's own channel analytics (views, watch time, click-through rate, subscriber count)

It only ever accesses data belonging to the single Google account that authorizes it (the channel owner's own account) and does not access any other user's data.
