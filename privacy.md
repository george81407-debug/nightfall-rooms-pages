# Privacy Policy — Nightfall Rooms Uploader

Last updated: 2026-07-14

Nightfall Rooms Uploader ("the App") is a private, internal automation tool built and used exclusively by the owner of the Nightfall Rooms YouTube channel to manage that channel's content. It is not distributed to, or intended for use by, any other individual or organization.

## What the App does

The App uses the YouTube Data API and YouTube Analytics API, via Google's OAuth 2.0 authorization, to:

- Upload videos to the operator's own YouTube channel
- Set video metadata (title, description, tags, privacy status, and synthetic-content disclosure)
- Read the operator's own channel analytics (e.g. views, watch time, click-through rate, subscriber count)

## Data accessed

The App only accesses data belonging to the single Google/YouTube account that authorizes it (the channel owner's own account). It does not access, request, or process data belonging to any other Google user.

## Data storage and sharing

- Authorization tokens and any data retrieved from the YouTube APIs are stored locally on the operator's own computer.
- No data is transmitted to, sold to, or shared with any third party.
- No data is used for advertising or any purpose other than operating the operator's own YouTube channel.

## Data retention

Locally stored data is retained only as long as needed to operate the channel and is deleted or overwritten as part of the operator's normal workflow.

## Revoking access

The account owner can revoke the App's access at any time via [Google Account permissions](https://myaccount.google.com/permissions).
